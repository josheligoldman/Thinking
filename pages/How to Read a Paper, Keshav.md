- # Notes
	- Read the paper in 3 passes
	- # Pass 1
		- Read Title, Abstract, Introduction
		- Read Section Headings
		- Read Conclusions
		- Check the references
		- Answer the 5 Cs
			- Category:
			- Context
			- Correctness
			- Contributions
			- Clarity
		- From this information, determine whether you want to read the paper
